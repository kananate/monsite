# Exemple de site Web
Ceci est un exemple de site Web réalisé dans le cadre de Git
, probablement le plus grand
Tutoriel Git pour débutants. 
Utiliser Git pour suivre les modifications de
vos projets, sauvegardez vos données, partagez votre travail avec d'autres et collaborez
avec des programmeurs et d'autres utilisateurs de Git.
Pour plus d'informations sur Git, consultez le
[documentation officielle de Git](https://git-scm.com/).
